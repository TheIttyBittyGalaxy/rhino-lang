## Setup


## Scripts

### Compiler

| Script  | Description                |
| ------- | -------------------------- |
| `build` | Build the interpreter      |
| `comp`  | Run the interpreter        |
| `debug` | Run the interpreter in gdb |
| `run`   | Run the rhino interpreter  |

### Test program

| Script  | Description            |
| ------- | ---------------------- |
| `btest` | Build the test program |
| `rtest` | Run the test program   |
